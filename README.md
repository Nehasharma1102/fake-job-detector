# 🔍 Fake Job Detector (Women Safety)

## 👩‍💻 Team: CodeNexa

### 💡 Problem
Many women fall victim to fake job offers, especially online and work-from-home scams.

### 🚀 Solution
We built an AI-powered system that detects whether a job posting is REAL or FAKE.

### 🤖 Technologies Used
- Python (Flask)
- Machine Learning
- TF-IDF Vectorization
- Logistic Regression
- HTML, CSS, JavaScript

### ⚙️ How It Works
1. User pastes job description
2. Text is processed using TF-IDF
3. ML model predicts fake or real
4. Rule-based logic improves accuracy
5. Result + confidence is displayed

### 🔥 Features
- Fake/Real detection
- Confidence score
- Suspicious keyword detection
- Clean UI

### 🛡️ Impact
Helps women stay safe from job scams and frauds.
